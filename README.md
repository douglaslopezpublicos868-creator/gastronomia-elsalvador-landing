# Sabores de El Salvador - Landing Page

## Descripción

Landing page desarrollada con HTML, CSS y JavaScript inspirada en la gastronomía de El Salvador.

## Características

- Diseño responsivo
- Navbar fija
- Menú hamburguesa
- Efecto scroll en navbar
- Validación de formulario con JavaScript
- Diseño con CSS Grid y Flexbox
- Publicación con GitHub Pages

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
